# My PDV SaaS V1.0.1 Comercial

Base comercial multiempresa criada a partir do ERP aprovado.

## Inclui
- Firebase do projeto My PDV
- Login com Firebase Authentication por e-mail/senha
- Cadastro de empresa/loja
- Dados separados por empresa em `empresas/{empresaId}/sistema/state`
- Planos Starter, Pro e Premium
- Painel Master para o e-mail principal
- Backup JSON por empresa
- PWA instalável

## Importante
Esta é a fundação SaaS. Antes de vender em escala, teste com 1 a 3 lojas beta.


Correção: API key Firebase ajustada para permitir cadastro/login por e-mail.
