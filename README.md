# My PDV SaaS V1.0.5 Comercial

Sistema completo para publicar em um repositório novo do GitHub Pages.

## Arquivos principais
- index.html
- app.js
- style.css
- manifest.json
- pwa.js
- service-worker.js
- icons/

## Firebase configurado
Projeto: my-pdv-85b1e
Authentication: e-mail/senha
Firestore: empresas/{empresaId}/sistema/state

## Publicação
1. Crie um repositório novo no GitHub.
2. Envie todos os arquivos extraídos deste ZIP.
3. Vá em Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Save.
7. Abra o link gerado e use Ctrl + F5.

## Observação
Esta versão usa cache novo v104 para evitar carregar versões antigas do PWA.
